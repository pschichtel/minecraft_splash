Minecraft Splashes for Flexile!
===============================

Description
-----------

This simple script replaces the normal right content box of Flexile
with flashing* random messages just like in your favorite game Minecraft!

* only works in browsers supporting @keyframes (current Mozilla and Webkit browsers)

Installation
------------

- Upload the splash folder onto you webserver
- Embed the the files splash/splash.css and splash/splash.js in your page header
- activate you right content box if not already active
- *DONE*!

Example embed code
------------------

<link rel="stylesheet" type="text/css" href="splash/splash.css">
<script type="text/javascript" href="splash/splash.js"></script>

This should be between <head> and </head>

Configuration
-------------

The messages can be changed inside the splash/splash.js
The messsages also support colorcodes in the typical format:

&a, &b, &c, ..., &e, &f
